Progress Bars

Progress Bars can be used to show the progress of any process or as a chart.

They are animated using CSS

Coloring is done with the CSS classes by using the color swatch colors

Custom Directory

This is a special directory for your custom SASS files.

Place your own custom files in this directory to protect them from being deleted during updates.

To add them to the generated CSS files, just add a simple line.

Let's say you have a custom _productlist.sass file. Then simply add

@import 'custom/productlist';

to your SCSS masterfile

Example

you have a file _productlist.sass in this directory.
To add this file in your scss file i.e. tvidoo-grid.scss simply put the following line
in the scss file:
<code class="css">@import 'custom/productlist';</code>
Where the best place for this line is, depends on your needs.
WARN: files within this directory are affected by the .gitignore file. If you want to add files to the repository you should consider adding them to one of the other directories within the structure.
